# ToHex

Very simple program that converts decimal to hex.

## Compalaition

make targets
| Target | Description |
|-------:|-------------|
| _(none)_ | Copys ToHex to /usr/bin/ and puts scripts in /usr/share/ToHex |
| remove | Removes ToHex from /usr/bin/ |

## Usage

Just use:
```
$ ToHex --help
$ ToHex 38
$ ToHex -term
```
to use anywhere.
